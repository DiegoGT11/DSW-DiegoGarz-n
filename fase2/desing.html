Conocimiento sobre etiquetas básicas de HTML, atributos y valores

El proposito central de este sitio web es fomentar el conocimiento de los conceptos básicos del HTML, en este sitio web se iran abordando disitintos conceptos que comprenden el desarrollo de sitios web como el uso de etiquetas básicas, atríbutos y valores. El aprendizaje de dichos conceptos permite al usuario fortalecer sus conocimientos y habilidades en el diseño de sitios web


HyperText Markup Language (HTML) consiste en un lenguaje utilizado para el desarrollo de sitios webs centrandose unicamente en la estrcutura del contenido de la página, este sistema funciona mediante el uso de etiquetas, atributos y valores los cuales le otorgaran una función disitnta al texto permiteiendo estrcuturar completamente el sitio web


Etiquetas básicas de HTML

El uso de etiquetas es escencial para poder comenzar a desarrollar un sitio web ya que permiten estrcucturar las bases de la pagina permitiendo organizar el encabezado, cuerpo, lineas de texto, tablas, etc.

Etiquetas

  • html: indica el comienzo del documento HTML
  • head: indica que empieza la cabecera de la página. En ella se suele poner el título title de la web, una descripción y otras informaciones relacionadas con el contenido de la página.
  • body: es el cuerpo de la página, donde va lo que se ve en el navegador al cargar una web. En el body van los textos, las imágenes y todos los contenidos de la web.
  • h1, h2, etc.: son los títulos o encabezados.
  • a: define los enlaces.
  • p: el texto dentro de esta etiqueta forma un párrafo.
  • img: imágenes.
  • ul: los textos dentro de esta etiqueta se estructuran en listas. Mediante el uso de li definimos cada guión dentro de la lista, y usando ol en lugar de ul tendremos listas ordenadas.
  • b y strong: se utilizan para resaltar el texto.
  • i: texto en cursiva.
Milan, V. (2019) Conceptos básicos de HTML. LaWebera.es. Recuperado de: https://www.lawebera.es/xhtml-css/conceptos-basicos-html.php

Atributos en HTML

Los atributos en HTML se consideran como un información adicional a la etiqueta para modificar o agregar alguna caracteristica al contenido. Los atributos tienen una forma correcta de escribirse, dentro de la etiqueta de apertura luego el simbolo igual (=) y entre comillas ("") el valor del atributo. Por otro lado, gran parte de los atributos son usados para darle estilo y estetica a la pagina, pero para ello existen softwares especiales para editar este apartado

Atributos

  • Id =” ”: Para identificar un elemento único.
  • class =” ”: Identificador múltiple.
  • align =” ”: Alineación de contenido.
  • border =” ”: Para darle borde al contenido.
  • style =” ”: Para darle un estilo al contenido.
  • background-color =” ”: Para color de fondo.
  • href =” ”: Para enlaces html.
  • height =” ”:Para determinar altura.
  • width =” ”:Para determinar ancho.
  • src=“ ”:Para imágenes.

Valores para atributos

Los valores son utilizados para definir a que medida se modificara el atributo, es decir los valores modificaran de cierta forma cada atributo, por ejemplo medida de una imágen, color de fonde o incluso para colocar un url para re direccionar la página

Atributos con sus respectivos valores

  • Id: su valor puede ser cualquier palabra que sirve para identificar un elemento.
  • align: alineación, sus valores pueden ser: center, right, left, justify.
  • border-style: estilo de borde, los valores disponibles son: solid, dotted, dashed, double, etc.
  • bgcolor: Los valores utilizados para este tributo pueden ser escritos en formato palabra, hexadecimal, RGB y demás
  • href: El valor es la dirección de una carpeta o URL
  • height y width: se utilizan valores de medida
